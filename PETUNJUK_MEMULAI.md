## ğŸ› Troubleshooting

### Problem: Website tidak terbuka
**Solusi:**
1. Pastikan XAMPP Apache sudah running
2. Cek URL di browser: `http://localhost/kaggleradya/`
3. Restart XAMPP jika perlu
4. Cek folder path: `C:\xampp1\htdocs\kaggleradya\`

### Problem: CSS tidak terbaca (tampilan berantakan)
**Solusi:**
1. Refresh browser (F5 atau Ctrl+R)
2. Clear cache browser (Ctrl+Shift+Delete)
3. Cek file `css/style.css` masih ada
4. Gunakan full path: `http://localhost/kaggleradya/index.html`

### Problem: Link tidak berfungsi
**Solusi:**
1. Cek folder `lessons/` ada di lokasi yang benar
2. Pastikan semua file HTML sudah di-download
3. Gunakan full URL bukan relative path
4. Cek spelling nama file
