Apa itu List?

List adalah struktur data yang menyimpan kumpulan elemen dalam urutan tertentu. List dapat berisi elemen dengan tipe data yang berbeda-beda. List ditulis dengan elemen-elemen dipisahkan oleh koma dan dikelilingi oleh tanda kurung siku [].

# Membuat list
buah = ["apel", "mangga", "jeruk"]
angka = [1, 2, 3, 4, 5]
campuran = [1, "dua", 3.0, True]
list_kosong = []

Mengakses Elemen List

Setiap elemen dalam list memiliki indeks (posisi). Indeks dimulai dari 0. Gunakan indeks untuk mengakses elemen tertentu.

buah = ["apel", "mangga", "jeruk", "pisang"]

print(buah[0]) # Output: apel (elemen pertama)
print(buah[1]) # Output: mangga (elemen kedua)
print(buah[-1]) # Output: pisang (elemen terakhir)
print(buah[-2]) # Output: jeruk (elemen kedua dari belakang)

Slicing List

Slicing memungkinkan Anda mengambil sebagian dari list dengan menentukan rentang indeks.

buah = ["apel", "mangga", "jeruk", "pisang", "nanas"]

print(buah[1:3]) # Output: ['mangga', 'jeruk']
print(buah[:3]) # Output: ['apel', 'mangga', 'jeruk']
print(buah[2:]) # Output: ['jeruk', 'pisang', 'nanas']
print(buah[::2]) # Output: ['apel', 'jeruk', 'nanas'] (setiap 2)

Mengubah Elemen List

Anda dapat mengubah nilai elemen dalam list dengan mengaksesnya melalui indeks.

buah = ["apel", "mangga", "jeruk"]

buah[1] = "anggur"
print(buah) # Output: ['apel', 'anggur', 'jeruk']

Menambah Elemen (Append)

Gunakan metode append() untuk menambah elemen baru di akhir list.

buah = ["apel", "mangga"]
buah.append("jeruk")
print(buah) # Output: ['apel', 'mangga', 'jeruk']

Menghapus Elemen

Ada beberapa cara untuk menghapus elemen dari list.

buah = ["apel", "mangga", "jeruk", "pisang"]

# Menggunakan remove() - menghapus berdasarkan nilai
buah.remove("mangga")
print(buah) # Output: ['apel', 'jeruk', 'pisang']

# Menggunakan pop() - menghapus berdasarkan indeks
elemen = buah.pop(1)
print(elemen) # Output: jeruk
print(buah) # Output: ['apel', 'pisang']

# Menggunakan del - menghapus elemen pada indeks
del buah[0]
print(buah) # Output: ['pisang']

Panjang List

Gunakan fungsi len() untuk mengetahui jumlah elemen dalam list.

buah = ["apel", "mangga", "jeruk"]
print(len(buah)) # Output: 3

Mengurutkan List

Gunakan metode sort() untuk mengurutkan elemen dalam list.

angka = [3, 1, 4, 1, 5, 9, 2, 6]
angka.sort()
print(angka) # Output: [1, 1, 2, 3, 4, 5, 6, 9]

buah = ["jeruk", "apel", "mangga"]
buah.sort()
print(buah) # Output: ['apel', 'jeruk', 'mangga']

Memeriksa Keberadaan Elemen

Gunakan operator in untuk memeriksa apakah elemen ada dalam list.

buah = ["apel", "mangga", "jeruk"]

print("apel" in buah) # Output: True
print("pisang" in buah) # Output: False
print("apel" not in buah) # Output: False

List Comprehension (Dasar)

List comprehension adalah cara singkat untuk membuat list baru berdasarkan list yang sudah ada.

angka = [1, 2, 3, 4, 5]
kuadrat = [x**2 for x in angka]
print(kuadrat) # Output: [1, 4, 9, 16, 25]

Ringkasan Pelajaran

  • List adalah kumpulan elemen yang terurut dan dapat diubah
  • Indeks dimulai dari 0
  • Indeks negatif dapat digunakan untuk mengakses elemen dari belakang
  • Slicing memungkinkan Anda mengambil bagian dari list
  • Metode seperti append(), remove(), pop() untuk memanipulasi list
  • sort() untuk mengurutkan list
  • Gunakan in untuk memeriksa keberadaan elemen
