Apa itu Library?

Library (atau module) adalah kumpulan kode yang telah ditulis sebelumnya yang dapat Anda gunakan untuk memperluas fungsionalitas program Anda. Python memiliki banyak library built-in dan dapat mengunduh library tambahan dari internet.

Import Library

Untuk menggunakan library, Anda perlu mengimpor (import) terlebih dahulu menggunakan kata kunci import.

# Import seluruh library
import math

# Import spesifik dari library
from math import pi, sqrt

# Import dengan alias
import numpy as np

Library Built-in: Math

Library math menyediakan fungsi-fungsi matematika.

import math

print(math.pi) # Output: 3.141592653589793
print(math.sqrt(16)) # Output: 4.0
print(math.ceil(4.3)) # Output: 5 (pembulatan ke atas)
print(math.floor(4.9)) # Output: 4 (pembulatan ke bawah)
print(math.pow(2, 3)) # Output: 8.0

Library Built-in: Random

Library random menyediakan fungsi untuk membuat angka atau memilih elemen secara acak.

import random

# Angka acak antara 0 dan 1
print(random.random())

# Angka acak bulat
print(random.randint(1, 10)) # Output: angka acak 1-10

# Memilih elemen acak dari list
pilihan = random.choice(["merah", "biru", "hijau"])
print(pilihan)

# Mengacak urutan list
angka = [1, 2, 3, 4, 5]
random.shuffle(angka)
print(angka) # Urutan berubah secara acak

Library Built-in: Datetime

Library datetime digunakan untuk bekerja dengan tanggal dan waktu.

from datetime import datetime, timedelta

# Tanggal dan waktu sekarang
sekarang = datetime.now()
print(sekarang) # Output: 2025-11-29 12:34:56.789123

# Membuat tanggal tertentu
tanggal = datetime(2025, 1, 1)
print(tanggal) # Output: 2025-01-01 00:00:00

# Menambah hari
besok = sekarang + timedelta(days=1)
print(besok)

# Format tanggal
print(sekarang.strftime("%d-%m-%Y")) # Output: 29-11-2025

Library PopulerExternal: NumPy

NumPy adalah library untuk komputasi numerik dengan array dan matriks. Harus diunduh terlebih dahulu.

import numpy as np

# Membuat array
arr = np.array([1, 2, 3, 4, 5])
print(arr)

# Array 2D
matrix = np.array([[1, 2, 3], [4, 5, 6]])
print(matrix)

# Operasi array
print(arr * 2) # Output: [ 2 4 6 8 10]
print(arr + arr) # Output: [ 2 4 6 8 10]
print(np.sum(arr)) # Output: 15
print(np.mean(arr)) # Output: 3.0

Library PopulerExternal: Pandas

Pandas adalah library untuk analisis dan manipulasi data.

import pandas as pd

# Membuat DataFrame
data = {
    "nama": ["Budi", "Ani", "Citra"],
    "umur": [20, 19, 21],
    "kota": ["Jakarta", "Bandung", "Surabaya"]
}

df = pd.DataFrame(data)
print(df)

# Akses kolom
print(df["nama"])

# Statistik
print(df["umur"].mean())

Mengunduh dan Menginstal Library

Untuk mengunduh library dari internet, gunakan pip (Python Package Manager).

# Di command line / terminal:
pip install numpy
pip install pandas
pip install requests

# Mengecek versi yang terinstal
pip show numpy

Library PopulerExternal: Requests

Requests adalah library untuk membuat HTTP requests (mengambil data dari internet).

import requests

# Mengambil data dari URL
response = requests.get("https://api.example.com/data")
print(response.status_code) # 200 artinya berhasil
print(response.json()) # Parse JSON response

Membuat Module Sendiri

Anda juga dapat membuat module/library sendiri.

# File: kalkulator.py
def tambah(a, b):
    return a + b

def kurang(a, b):
    return a - b

# File: main.py
import kalkulator

print(kalkulator.tambah(5, 3)) # Output: 8
print(kalkulator.kurang(5, 3)) # Output: 2

Try-Except untuk Error Handling

Ketika menggunakan library, mungkin terjadi error. Gunakan try-except untuk menangani error.

try:
    import non_existent_library
except ImportError:
    print("Library tidak ditemukan")

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Tidak dapat membagi dengan 0")
except Exception as e:
    print(f"Error terjadi: {e}")
