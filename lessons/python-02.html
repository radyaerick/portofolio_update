Apa itu Fungsi?

Fungsi adalah blok kode yang dapat digunakan kembali untuk melakukan tugas tertentu. Fungsi membantu membuat kode lebih terorganisir, mudah dibaca, dan dapat digunakan kembali.

Mendeklarasikan Fungsi

Untuk membuat fungsi, kita menggunakan kata kunci def diikuti dengan nama fungsi dan tanda kurung.

# Fungsi tanpa parameter
def salam():
    print("Halo, selamat datang!")

# Memanggil fungsi
salam() # Output: Halo, selamat datang!

Fungsi dengan Parameter

Parameter adalah nilai yang diterima oleh fungsi. Dengan parameter, fungsi dapat beroperasi pada data yang berbeda.

# Fungsi dengan satu parameter
def salam(nama):
    print("Halo, " + nama)

salam("Budi") # Output: Halo, Budi
salam("Ani") # Output: Halo, Ani

# Fungsi dengan multiple parameter
def tambah(a, b):
    return a + b

hasil = tambah(5, 3) # hasil = 8

Return Statement

Pernyataan return digunakan untuk mengembalikan nilai dari fungsi. Ketika fungsi mencapai return, eksekusi fungsi berhenti dan nilai dikembalikan ke pemanggil.

def luas_lingkaran(jari_jari):
    import math
    return math.pi * jari_jari ** 2

hasil = luas_lingkaran(5)
print(hasil) # Output: 78.53981633974483

Scope (Cakupan Variabel)

Scope menentukan di mana variabel dapat diakses. Variabel yang didefinisikan di dalam fungsi hanya dapat diakses di dalam fungsi tersebut.

def test_scope():
    x = 10 # Variabel lokal
    print(x) # Bisa diakses

test_scope() # Output: 10
print(x) # Error: x tidak terdefinisi di luar fungsi

Mendapatkan Bantuan (Help)

Python memiliki sistem bantuan bawaan yang dapat membantu Anda memahami fungsi dan modul. Gunakan fungsi help() atau tanda ? di Jupyter Notebook.

# Mendapatkan bantuan tentang fungsi
help(print)
help(len)

# Atau gunakan ? di Jupyter
?print

Default Parameter

Anda dapat memberikan nilai default untuk parameter. Jika parameter tidak diberikan saat memanggil fungsi, nilai default akan digunakan.

def salam(nama, sapaan="Halo"):
    print(sapaan + ", " + nama)

salam("Budi") # Output: Halo, Budi
salam("Ani", "Selamat pagi") # Output: Selamat pagi, Ani

Fungsi Built-in Python

Python menyediakan banyak fungsi bawaan yang siap digunakan tanpa harus dibuat sendiri.

  • len() - Mengembalikan panjang/jumlah elemen
  • max() - Mengembalikan nilai maksimum
  • min() - Mengembalikan nilai minimum
  • sum() - Mengembalikan jumlah elemen
  • type() - Mengembalikan tipe data
  • str() - Mengubah ke string
  • int() - Mengubah ke integer
  • float() - Mengubah ke float
üí° Tips:

Gunakan nama fungsi yang deskriptif agar kode lebih mudah dipahami. Hindari menggunakan nama fungsi bawaan Python untuk fungsi Anda sendiri.

Ringkasan Pelajaran

  • Fungsi adalah blok kode yang dapat digunakan kembali
  • Deklarasikan fungsi dengan def
  • Parameter memungkinkan fungsi menerima nilai
  • Return statement mengembalikan nilai dari fungsi
  • Gunakan help() untuk mendapatkan bantuan
  • Parameter dapat memiliki nilai default
