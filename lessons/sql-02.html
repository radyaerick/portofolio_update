Select, From & Where

Pelajaran 2: Tiga Perintah Dasar SQL

Perintah SELECT

Perintah SELECT digunakan untuk memilih kolom-kolom mana saja yang ingin Anda tampilkan dari database.

-- Memilih semua kolom
SELECT * FROM siswa;

-- Memilih kolom tertentu
SELECT nama, umur FROM siswa;

-- Memilih satu kolom
SELECT nama FROM siswa;

Perintah FROM

Perintah FROM menentukan tabel mana yang ingin Anda ambil data. Setiap query harus menentukan sumber data dengan FROM.

-- Mengambil dari tabel siswa
SELECT * FROM siswa;

-- Mengambil dari tabel dengan nama spesifik (dataset.tabel)
SELECT * FROM `project.dataset.siswa`;

Perintah WHERE

Perintah WHERE digunakan untuk memfilter data berdasarkan kondisi tertentu. Hanya baris yang memenuhi kondisi yang akan ditampilkan.

-- Mengambil siswa dengan umur lebih dari 20
SELECT nama, umur FROM siswa WHERE umur > 20;

-- Mengambil siswa dari kota Jakarta
SELECT * FROM siswa WHERE kota = 'Jakarta';

-- Mengambil siswa dengan umur sama dengan 20
SELECT * FROM siswa WHERE umur = 20;

Operator Perbandingan dalam WHERE

Ada berbagai operator yang dapat digunakan dalam klausa WHERE:

  • = - Sama dengan
  • != atau <> - Tidak sama dengan
  • > - Lebih besar dari
  • < - Lebih kecil dari
  • >= - Lebih besar atau sama dengan
  • <= - Lebih kecil atau sama dengan
-- Umur tidak sama dengan 20
SELECT * FROM siswa WHERE umur != 20;

-- Umur lebih besar atau sama dengan 20
SELECT * FROM siswa WHERE umur >= 20;

-- Umur kurang dari 21
SELECT * FROM siswa WHERE umur < 21;

Operator Logika: AND, OR, NOT

Operator logika digunakan untuk menggabungkan beberapa kondisi.

-- AND: Kedua kondisi harus benar
SELECT * FROM siswa WHERE umur > 19 AND kota = 'Jakarta';

-- OR: Salah satu kondisi harus benar
SELECT * FROM siswa WHERE kota = 'Jakarta' OR kota = 'Bandung';

-- NOT: Kondisi harus salah
SELECT * FROM siswa WHERE NOT kota = 'Jakarta';

Operator IN

Operator IN digunakan untuk mencocokkan nilai dalam daftar.

-- Siswa dari Jakarta, Bandung, atau Surabaya
SELECT * FROM siswa
WHERE kota IN ('Jakarta', 'Bandung', 'Surabaya');

-- Siswa dengan umur 19 atau 21
SELECT * FROM siswa WHERE umur IN (19, 21);

Operator BETWEEN

Operator BETWEEN digunakan untuk memilih nilai dalam rentang tertentu.

-- Siswa dengan umur antara 19 dan 21
SELECT * FROM siswa WHERE umur BETWEEN 19 AND 21;

-- Ini sama dengan:
SELECT * FROM siswa WHERE umur >= 19 AND umur <= 21;

Operator LIKE

Operator LIKE digunakan untuk mencari pola dalam string.

-- Nama yang dimulai dengan 'B'
SELECT * FROM siswa WHERE nama LIKE 'B%';

-- Nama yang berakhir dengan 'i'
SELECT * FROM siswa WHERE nama LIKE '%i';

-- Nama yang mengandung 'ud'
SELECT * FROM siswa WHERE nama LIKE '%ud%';

-- Nama dengan panjang 3 karakter
SELECT * FROM siswa WHERE nama LIKE '___';

Operator NULL

NULL adalah nilai khusus yang menandakan tidak ada data. Gunakan IS NULL atau IS NOT NULL untuk memeriksa NULL.

-- Data yang tidak memiliki nilai kota
SELECT * FROM siswa WHERE kota IS NULL;

-- Data yang memiliki nilai kota
SELECT * FROM siswa WHERE kota IS NOT NULL;

Kombinasi SELECT, FROM, dan WHERE

Berikut adalah contoh query yang menggabungkan ketiga perintah:

-- Ambil nama dan umur dari siswa yang berumur lebih dari 19 tahun
SELECT nama, umur
FROM siswa
WHERE umur > 19;

-- Ambil semua data siswa dari Jakarta atau Bandung
SELECT *
FROM siswa
WHERE kota IN ('Jakarta', 'Bandung');

-- Ambil nama dan kota dari siswa dengan umur antara 19 dan 21
SELECT nama, kota
FROM siswa
WHERE umur BETWEEN 19 AND 21
AND kota IS NOT NULL;
üí° Tips:

Klausa WHERE sangat penting untuk memfilter data. Gunakan kondisi yang tepat untuk mendapatkan data yang Anda butuhkan. Jangan lupa menggunakan tanda kutip tunggal untuk string dalam WHERE.

Ringkasan Pelajaran

  • SELECT memilih kolom yang ingin ditampilkan
  • FROM menentukan tabel sumber data
  • WHERE memfilter baris berdasarkan kondisi
  • Operator perbandingan: =, !=, >, <, >=, <=
  • Operator logika: AND, OR, NOT
  • Operator khusus: IN, BETWEEN, LIKE, IS NULL
  • Kombinasi SELECT, FROM, WHERE membuat query yang powerful
