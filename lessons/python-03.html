Pelajaran 3: Pengambilan Keputusan dalam Program

Apa itu Boolean?

Boolean adalah tipe data yang hanya memiliki dua nilai: True (Benar) atau False (Salah). Boolean sering digunakan dalam kondisional dan perulangan untuk membuat keputusan dalam program.

# Variabel Boolean
is_raining = True
is_sunny = False

print(is_raining) # Output: True
print(is_sunny) # Output: False

Operator Perbandingan

Operator perbandingan digunakan untuk membandingkan dua nilai dan menghasilkan nilai Boolean.

a = 10
b = 5

print(a > b) # True (a lebih besar dari b)
print(a < b) # False (a lebih kecil dari b)
print(a == b) # False (a sama dengan b)
print(a != b) # True (a tidak sama dengan b)
print(a >= b) # True (a lebih besar atau sama dengan b)
print(a <= b) # False (a lebih kecil atau sama dengan b)

Operator Logika

Operator logika digunakan untuk menggabungkan dua atau lebih kondisi Boolean.

  • and - Benar jika semua kondisi benar
  • or - Benar jika salah satu kondisi benar
  • not - Membalikkan nilai Boolean
x = 10
y = 5

print(x > 5 and y > 3) # True (kedua kondisi benar)
print(x > 5 and y > 10) # False (salah satu kondisi salah)
print(x > 5 or y > 10) # True (salah satu kondisi benar)
print(not(x > 5)) # False (membalikkan True menjadi False)

Pernyataan If

Pernyataan if memungkinkan Anda menjalankan blok kode hanya jika kondisi tertentu terpenuhi.

umur = 18

if umur >= 18:
    print("Anda sudah dewasa")

Pernyataan If-Else

Pernyataan if-else memungkinkan Anda menjalankan satu blok kode jika kondisi benar, dan blok kode lain jika kondisi salah.

umur = 15

if umur >= 18:
    print("Anda sudah dewasa")
else:
    print("Anda masih anak-anak")

Pernyataan If-Elif-Else

Pernyataan elif (else if) memungkinkan Anda menguji kondisi tambahan jika kondisi pertama salah.

nilai = 85

if nilai >= 90:
    print("Nilai A - Sangat Baik")
elif nilai >= 80:
    print("Nilai B - Baik")
elif nilai >= 70:
    print("Nilai C - Cukup")
else:
    print("Nilai D - Kurang")

Nested If (If Bersarang)

Anda dapat menempatkan pernyataan if di dalam pernyataan if lain.

umur = 20
memiliki_sim = True

if umur >= 18:
    if memiliki_sim:
        print("Anda boleh berkendara")
    else:
        print("Anda perlu SIM terlebih dahulu")
else:
    print("Anda terlalu muda untuk berkendara")

Ternary Operator (Operator Ternari)

Ternary operator adalah cara singkat untuk menulis pernyataan if-else dalam satu baris.

umur = 20
status = "Dewasa" if umur >= 18 else "Anak-anak"
print(status) # Output: Dewasa
üí° Tips:

Indentasi (spasi di awal baris) sangat penting dalam Python. Blok kode yang termasuk dalam if atau else harus memiliki indentasi yang konsisten.

Ringkasan Pelajaran

  • Boolean adalah tipe data dengan nilai True atau False
  • Operator perbandingan (>, <, ==, !=, >=, <=) menghasilkan Boolean
  • Operator logika (and, or, not) menggabungkan kondisi Boolean
  • Pernyataan if digunakan untuk membuat keputusan
  • Elif memungkinkan menguji kondisi tambahan
  • Indentasi adalah bagian penting dari sintaks Python
