Apa itu SQL?

SQL (Structured Query Language) adalah bahasa pemrograman yang dirancang khusus untuk mengelola dan mengambil data dari database. SQL memungkinkan Anda untuk membuat, membaca, memperbarui, dan menghapus data dalam database dengan cara yang efisien.

Apa itu Database?

Database adalah kumpulan data yang terorganisir dalam bentuk tabel. Setiap tabel terdiri dari baris (records) dan kolom (fields). Database digunakan untuk menyimpan informasi dalam jumlah besar dengan cara yang terstruktur dan mudah diakses.

Apa itu BigQuery?

Google BigQuery adalah layanan cloud computing dari Google yang memungkinkan Anda menganalisis data dalam skala besar dengan cepat. BigQuery menyimpan data dalam tabel dan memungkinkan Anda menggunakan SQL untuk menganalisisnya. BigQuery sangat cocok untuk analisis data besar (big data).

Struktur Database

Database terdiri dari satu atau lebih tabel. Setiap tabel memiliki struktur berikut:

Tabel: siswa
---------------------------------
| id | nama | umur | kota |
---------------------------------
| 1 | Budi | 20 | Jakarta |
| 2 | Ani | 19 | Bandung |
| 3 | Citra | 21 | Surabaya |
---------------------------------

Kolom: id, nama, umur, kota
Baris: data setiap siswa

Tipe Data dalam SQL

Database mendukung berbagai tipe data:

  • INTEGER - Bilangan bulat (1, 100, -50)
  • VARCHAR/STRING - Teks (nama, alamat)
  • DATE - Tanggal (2025-01-15)
  • FLOAT/DECIMAL - Bilangan desimal (3.14, 100.50)
  • BOOLEAN - True atau False

Struktur Query SQL

Query SQL memiliki struktur dasar yang disebut SELECT statement. Ini adalah query paling dasar untuk mengambil data dari database.

SELECT nama, umur
FROM siswa
WHERE umur > 20
ORDER BY nama;

Menggunakan BigQuery

Untuk menggunakan BigQuery:

  1. Buka https://console.cloud.google.com/
  2. Buat project baru atau pilih project yang sudah ada
  3. Aktifkan BigQuery API
  4. Buka BigQuery console
  5. Tulis query SQL di editor
  6. Klik "Run" untuk menjalankan query

Datasets Publik BigQuery

BigQuery menyediakan dataset publik yang dapat Anda gunakan untuk belajar. Beberapa dataset populer:

  • bigquery-public-data:world_bank_intl_debt - Data utang internasional
  • bigquery-public-data:hacker_news - Data Hacker News
  • bigquery-public-data:github_repos - Data repository GitHub
  • bigquery-public-data:wikipedia - Data Wikipedia

Query Pertama Anda

Mari coba query pertama dengan dataset publik:

SELECT
    COUNT(*) as total_records
FROM
    `bigquery-public-data.hacker_news.stories`
LIMIT 10;

Query di atas akan menghitung jumlah total records dalam tabel stories dataset Hacker News.

Dasar-dasar Syntax SQL

Beberapa hal penting dalam syntax SQL:

  • SQL tidak case-sensitive (SELECT = select), tetapi case-sensitive untuk string
  • Setiap statement SQL berakhir dengan semicolon (;)
  • String harus diapit oleh tanda kutip tunggal ('nilai')
  • Whitespace (spasi, tab, newline) diabaikan dalam SQL
  • Komentar dapat ditambahkan dengan -- atau /* */
-- Ini adalah komentar baris tunggal
SELECT nama FROM siswa; /* Ini adalah
komentar multi-baris */

Ringkasan Pelajaran

  • SQL adalah bahasa untuk mengelola database
  • Database terdiri dari tabel dengan baris dan kolom
  • BigQuery adalah platform cloud untuk analisis data
  • Query dasar menggunakan SELECT statement
  • BigQuery menyediakan dataset publik untuk belajar
  • SQL memiliki syntax yang perlu diikuti dengan baik
  • Komentar dapat ditambahkan ke query SQL
