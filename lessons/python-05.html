Apa itu Loop?

Loop (perulangan) digunakan untuk menjalankan blok kode berulang kali. Ada dua jenis loop dalam Python: for dan while.

For Loop

For loop digunakan untuk mengulangi blok kode untuk setiap elemen dalam urutan (seperti list, string, atau range).

# Loop melalui list
buah = ["apel", "mangga", "jeruk"]
for buah_item in buah:
    print(buah_item)

# Output:
# apel
# mangga
# jeruk

Range Function

Fungsi range() menghasilkan urutan angka. Range sangat berguna dalam for loop.

# Range dari 0 sampai 4 (tidak termasuk 5)
for i in range(5):
    print(i)

# Output: 0, 1, 2, 3, 4

# Range dari 2 sampai 8 dengan step 2
for i in range(2, 8, 2):
    print(i)

# Output: 2, 4, 6

Enumerate Function

Gunakan enumerate() jika Anda membutuhkan indeks dan elemen sekaligus.

buah = ["apel", "mangga", "jeruk"]

for indeks, buah_item in enumerate(buah):
    print(indeks, buah_item)

# Output:
# 0 apel
# 1 mangga
# 2 jeruk

While Loop

While loop menjalankan blok kode selama kondisi bernilai True.

i = 0
while i < 5:
    print(i)
    i = i + 1

# Output: 0, 1, 2, 3, 4

Break dan Continue

break menghentikan loop, sementara continue melewati iterasi saat ini dan melanjutkan ke iterasi berikutnya.

# Menggunakan break
for i in range(10):
    if i == 5:
        break # Keluar dari loop
    print(i)

# Output: 0, 1, 2, 3, 4

# Menggunakan continue
for i in range(5):
    if i == 2:
        continue # Lewati 2
    print(i)

# Output: 0, 1, 3, 4

List Comprehension

List comprehension adalah cara ringkas untuk membuat list baru. Ini menggabungkan for loop dan list creation dalam satu baris.

# Membuat list kuadrat dari 1-5
kuadrat = [x**2 for x in range(1, 6)]
print(kuadrat) # Output: [1, 4, 9, 16, 25]

# Dengan kondisi if
angka_genap = [x for x in range(10) if x % 2 == 0]
print(angka_genap) # Output: [0, 2, 4, 6, 8]

# Dari list yang sudah ada
buah = ["apel", "mangga", "jeruk"]
buah_besar = [b.upper() for b in buah]
print(buah_besar) # Output: ['APEL', 'MANGGA', 'JERUK']

Nested Loop

Anda dapat menempatkan loop di dalam loop lain untuk membuat perulangan bersarang.

# Membuat tabel perkalian
for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i} x {j} = {i*j}")

# Output:
# 1 x 1 = 1
# 1 x 2 = 2
# ... dan seterusnya

For-Else dan While-Else

Blok else dalam loop akan dijalankan jika loop selesai dengan normal (tidak menggunakan break).

for i in range(5):
    print(i)
else:
    print("Loop selesai!")

# Output:
# 0
# 1
# 2
# 3
# 4
# Loop selesai!
üí° Tips:

List comprehension lebih cepat dan lebih Pythonic daripada menggunakan loop untuk membuat list. Gunakan list comprehension ketika memungkinkan untuk membuat kode lebih efisien dan mudah dibaca.

Ringkasan Pelajaran

  • For loop digunakan untuk perulangan dengan jumlah iterasi yang diketahui
  • While loop digunakan untuk perulangan dengan kondisi
  • Break menghentikan loop, continue melewati iterasi
  • range() menghasilkan urutan angka
  • enumerate() memberikan indeks dan elemen
  • List comprehension adalah cara ringkas membuat list
  • Nested loop memungkinkan perulangan bersarang
