Apa itu String?

String adalah tipe data yang mewakili teks. String dapat dibuat menggunakan tanda kutip tunggal (') atau tanda kutip ganda (").

nama = "Budi"
pesan = 'Halo, dunia!'
multiline = """Ini adalah
string yang
memiliki banyak baris"""

Operasi pada String

String dapat dioperasikan dengan berbagai cara di Python.

# Penggabungan (concatenation)
nama_depan = "Budi"
nama_belakang = "Santoso"
nama_lengkap = nama_depan + " " + nama_belakang
print(nama_lengkap) # Output: Budi Santoso

# Pengulangan
print("Ha" * 3) # Output: HaHaHa

# Panjang string
print(len("Budi")) # Output: 4

Indexing dan Slicing String

String dapat diakses seperti list menggunakan indeks dan slicing.

text = "Python"
print(text[0]) # Output: P (karakter pertama)
print(text[2]) # Output: t
print(text[-1]) # Output: n (karakter terakhir)
print(text[1:4]) # Output: yth
print(text[:3]) # Output: Pyt

String Methods (Metode String)

Python menyediakan banyak metode untuk memanipulasi string.

text = " Halo Dunia "

# Upper dan Lower
print(text.upper()) # Output: HALO DUNIA
print(text.lower()) # Output: halo dunia

# Strip (menghapus spasi di awal dan akhir)
print(text.strip()) # Output: Halo Dunia

# Replace (mengganti)
print(text.replace("Dunia", "Python")) # Output: Halo Python

# Split (memisahkan)
kata = "Budi Santoso Wijaya".split()
print(kata) # Output: ['Budi', 'Santoso', 'Wijaya']

# Join (menggabungkan)
hasil = ", ".join(kata)
print(hasil) # Output: Budi, Santoso, Wijaya

# Find (mencari)
print(text.find("Dunia")) # Output: 7

# Startswith dan Endswith
print("Python".startswith("Py")) # Output: True
print("Python".endswith("on")) # Output: True

String Formatting

Ada beberapa cara untuk memformat string di Python.

nama = "Budi"
umur = 25

# Menggunakan % (cara lama)
print("Nama saya %s dan umur %d" % (nama, umur))

# Menggunakan .format()
print("Nama saya {} dan umur {}".format(nama, umur))

# Menggunakan f-string (cara modern)
print(f"Nama saya {nama} dan umur {umur}")

# Dengan ekspresi di f-string
nilai = 85
print(f"Nilai saya adalah {nilai} dan grade saya adalah {'A' if nilai >= 90 else 'B'}")

Apa itu Dictionary?

Dictionary adalah koleksi data yang tidak terurut, dapat diubah, dan dipasangkan antara key (kunci) dan value (nilai). Dictionary ditulis dengan kurung kurawal {}.

# Membuat dictionary
siswa = {
    "nama": "Budi",
    "umur": 20,
    "kota": "Jakarta"
}

# Dictionary kosong
data = {}

Mengakses Elemen Dictionary

Akses elemen dictionary menggunakan key, bukan indeks.

siswa = {"nama": "Budi", "umur": 20, "kota": "Jakarta"}

print(siswa["nama"]) # Output: Budi
print(siswa["umur"]) # Output: 20
print(siswa.get("kota")) # Output: Jakarta

# Menggunakan get() dengan default value
print(siswa.get("provinsi", "Tidak ada")) # Output: Tidak ada

Mengubah dan Menambah Elemen

Anda dapat mengubah atau menambah elemen dalam dictionary.

siswa = {"nama": "Budi", "umur": 20}

# Mengubah nilai
siswa["umur"] = 21

# Menambah elemen baru
siswa["kota"] = "Jakarta"

print(siswa) # Output: {'nama': 'Budi', 'umur': 21, 'kota': 'Jakarta'}

Menghapus Elemen

Gunakan del atau metode pop() untuk menghapus elemen.

siswa = {"nama": "Budi", "umur": 20, "kota": "Jakarta"}

# Menggunakan del
del siswa["kota"]

# Menggunakan pop()
umur = siswa.pop("umur")

print(siswa) # Output: {'nama': 'Budi'}

Loop dalam Dictionary

Anda dapat melakukan loop untuk mengakses semua elemen dalam dictionary.

siswa = {"nama": "Budi", "umur": 20, "kota": "Jakarta"}

# Loop keys
for key in siswa:
    print(key)

# Loop values
for value in siswa.values():
    print(value)

# Loop key dan value
for key, value in siswa.items():
    print(f"{key}: {value}")

Dictionary Methods

Dictionary memiliki beberapa metode yang berguna.

siswa = {"nama": "Budi", "umur": 20}

# Keys
print(siswa.keys()) # Output: dict_keys(['nama', 'umur'])

# Values
print(siswa.values()) # Output: dict_values(['Budi', 20])

# Items
print(siswa.items()) # Output: dict_items([('nama', 'Budi'), ('umur', 20)])

# Update
siswa.update({"kota": "Jakarta"})

# Clear (menghapus semua)
siswa.clear()
print(siswa) # Output: {}

Nested Dictionary

Dictionary dapat berisi dictionary lain di dalamnya.

kelas = {
    "siswa1": {"nama": "Budi", "umur": 20},
    "siswa2": {"nama": "Ani", "umur": 19}
}

print(kelas["siswa1"]["nama"]) # Output: Budi
üí° Tips:

String adalah immutable (tidak dapat diubah), tetapi dictionary adalah mutable (dapat diubah). Gunakan f-string untuk string formatting yang modern dan mudah dibaca.

Ringkasan Pelajaran

  • String adalah tipe data yang mewakili teks
  • String dapat diakses dengan indexing dan slicing
  • Ada banyak metode string untuk memanipulasi teks
  • String formatting dapat dilakukan dengan %, .format(), atau f-string
  • Dictionary menyimpan pasangan key-value
  • Dictionary dapat diakses, diubah, dan dihapus dengan mudah
  • Dictionary dapat di-loop untuk mengakses semua elemen
