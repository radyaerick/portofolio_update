Pengenalan Dasar Python

Python adalah bahasa pemrograman yang sangat populer, khususnya dalam bidang data science. Pada pelajaran pertama ini, kita akan mempelajari dasar-dasar Python termasuk sintaks, variabel, dan tipe data angka.

Apa itu Python?

Python adalah bahasa pemrograman yang mudah dipelajari namun sangat powerful. Dibuat oleh Guido van Rossum pada tahun 1991, Python dirancang untuk kode yang mudah dibaca dan ditulis. Python menjadi bahasa pilihan untuk data science karena memiliki banyak library yang powerful seperti NumPy, Pandas, dan Scikit-Learn.

Sintaks Dasar Python

Sintaks adalah aturan penulisan kode dalam bahasa pemrograman. Python memiliki sintaks yang sangat sederhana dan mudah dipahami.

# Ini adalah komentar
print("Halo, dunia!") # Menampilkan teks

Variabel dan Tipe Data

Variabel adalah tempat untuk menyimpan nilai. Python adalah bahasa yang dinamis, yang berarti Anda tidak perlu mendeklarasikan tipe data sebelum menggunakan variabel.

# Membuat variabel
nama = "Budi" # String (teks)
umur = 25 # Integer (bilangan bulat)
tinggi = 175.5 # Float (bilangan desimal)

Angka (Numbers) dalam Python

Python mendukung dua tipe angka utama: integers (bilangan bulat) dan floats (bilangan desimal).

# Integers
a = 10
b = -5

# Floats
c = 3.14
d = -2.5

# Operasi matematika
hasil_penjumlahan = a + c # 13.14
hasil_pengurangan = a - b # 15
hasil_perkalian = a * b # -50
hasil_pembagian = a / c # 3.18...
hasil_pangkat = a ** 2 # 100 (10 pangkat 2)
hasil_sisa = a % 3 # 1 (sisa pembagian)

Output dan Input

Untuk menampilkan nilai atau hasil, kita menggunakan fungsi print(). Untuk menerima input dari pengguna, kita menggunakan fungsi input().

# Menampilkan output
print("Saya sedang belajar Python")
print(10 + 20)

# Menerima input
nama = input("Siapa nama Anda? ")
print("Halo, " + nama)
